## **使用方式**
### **1. 命令面板**
1. 打开命令面板（快捷键：`Ctrl+Shift+P`）。
2. 输入以下命令之一：
   - `Parentheses Align: Left`
   - `Parentheses Align: Center`
   - `Parentheses Align: Right`
3. 执行命令后，当前选中文本或光标所在行的括号内容将自动对齐。

### **2. 快捷键**
| 功能       |  快捷键  |
|----------- |---------|
| 左对齐     | `Alt+L` |
| 居中对齐   | `Alt+C` |
| 右对齐     | `Alt+R` |
